# Quiz Management System

## 📌 Project Overview

The **Quiz Management System** is a web-based application designed to create, manage, and conduct quizzes online. It helps administrators add questions, manage quizzes, and automatically evaluate results while allowing users or students to take quizzes and view their scores.

This system simplifies the process of conducting online tests and improves efficiency by automating quiz management and result calculation. Online quiz systems are widely used in educational platforms to conduct assessments and track student performance. ([GitHub][1])

---

## 🚀 Features

* User registration and login
* Admin panel to manage quizzes
* Add, update, and delete questions
* Multiple choice questions (MCQs)
* Automatic score calculation
* Instant result display
* User-friendly interface
* Secure authentication system

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/sairamareddy2/Quiz-management-system.git
```

2. Navigate to the project folder

```bash
cd Quiz-management-system
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

## 📊 How It Works

1. Admin logs into the system.
2. Admin creates quizzes and adds questions.
3. Users register and log in.
4. Users take quizzes.
5. System automatically calculates and displays results.
